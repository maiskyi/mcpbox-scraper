# Smithery scraper

## Installation

```bash
pip install -r requirements.txt
```

## Setup .env

### Setup API key
```env
SMITHERY_API_KEY=...
```

### Setup update frequency in hours
```env
SMITHERY_UPDATE_FREQ=72
```


## Usage
### Run:
```
python main.py
```

Results will be saved to in ``result_list.json`` as list and ``result.json`` file as map
